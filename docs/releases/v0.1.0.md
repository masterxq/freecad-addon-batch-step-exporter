## Batch STEP Exporter v0.1.0

Initial public release.

### Highlights

- New FreeCAD workbench: Batch STEP Exporter
- Batch parameter iteration from Spreadsheet aliases
- Cartesian product export over multiple parameter rows
- Automatic recompute for each iteration
- Export of each PartDesign body to individual STEP files
- Folder naming based on parameter combinations (for example: `te_4_w_600_h_1800_d_500`)
- STEP naming format: `Iteration_Project_Part.step`
- CSV summary export and iteration manifest files
- Included screenshots and bilingual documentation (English + German)

### Typical Use Cases

- Publish one CAD project in many parameter variants
- Generate print-ready sets of model variants with minimal manual work

### Notes

- Tested with FreeCAD 1.1.1
- User module path for this setup: `~/.local/share/FreeCAD/v1-1/Mod`
